📢 具身智能科研实习招募

我们正在寻找对具身智能（Embodied AI）充满热情的科研实习生，加入我们正在开展的多项机器人智能项目，包括但不限于仿真训练、真实机器人实验、强化学习与多模态感知。

🧠 研究方向

  • 机器人任务规划与动作控制
  • 模拟环境中的强化学习训练（Isaac Gym、MuJoCo等）
  • 真实机器人系统（Unitree、UR机械臂、灵巧手等）上的实地操作与数据采集
  • Vision-Language-Action 模型在具身智能场景中的集成与探索

✅ 我们希望你具备以下之一或多项经验

  • 机器人平台（如 ROS, Unitree, UR, Franka 等）实际操作经验
  • 强化学习
  • 仿真建模经验（Isaac Sim、Blender + Python 脚本）
  • 熟悉 PyTorch 和深度学习，有阅读顶会论文经验

📍 实习信息

  • 线下实习将提供生活补助
  • 实习时长至少3个月（更长优先）

🎓 适合对象

  • 在读本科/硕士/博士生（计算机、自动化、机器人等专业）
  • 计划深入科研或希望积累项目经验者
  • 有意向参与发表或申请研究型学位者尤佳
